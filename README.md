📝 To-Do List Web App
📌 Project Explanation

This is a simple To-Do List web application where users can:

Add new tasks
Edit existing tasks
Delete tasks

It helps users manage their daily tasks easily. The app updates instantly when you interact with it (no page reload needed).

⚙️ Tech Stack Used
HTML → Structure of the webpage
CSS → Styling and layout
JavaScript → Functionality (add, edit, delete tasks)
▶️ How to Run the Project
Download or clone the project files
Open the project folder
Double-click the index.html file
(or right-click → open with browser)

That’s it — the app will run in your browser 

✨ Features
Add task
Edit task
Delete task
Input validation (no empty task allowed)
